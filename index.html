📂 Cara Update Data Setiap Tahun

  1. Terima file Excel Verval tahun baru dari Bidang Data · Contoh nama file: KALBAR2026.xlsx
  2. Klik tombol "Upload Excel" di bagian atas halaman ini (atau seret file ke area abu‑abu)
  3. Dasbor membaca otomatis sheet: kpop, kpot total, prod, kprod ternak besar&kecil
  4. Semua grafik, tabel, dan analisis disparitas langsung diperbarui dalam 3–5 detik
  5. Tidak perlu instalasi aplikasi, tidak perlu internet khusus — berjalan sepenuhnya di browser

🔧 Format Excel yang Dibutuhkan

  • Sheet kpop: Baris = Kabupaten/Kota · Kolom = Jenis Ternak × Tahun · Baris terakhir = TOTAL
  • Sheet kpot total: Sama seperti kpop, berisi data pemotongan total per jenis ternak
  • Sheet prod: Ringkasan produksi (daging, telur, susu) per jenis ternak per tahun
  • Sheet kprod ternak besar&kecil 20XX: Detail produksi ternak besar & kecil per kabupaten
  • Nama-nama kabupaten harus konsisten dengan tahun sebelumnya (14 kab/kota Kalbar)
  • Jika BPS mengubah format sheet, upload ke Claude untuk adaptasi template parser

📝 Cara Update Laporan Word Naratif (via Claude)

  1. Upload file Excel baru ke claude.ai
  2. Ketik: "Update laporan Word analisis peternakan Kalbar dengan data terbaru dari file ini"
  3. Claude akan membaca data, menghitung semua indikator, dan menulis narasi analisis baru
  4. Laporan Word siap diunduh dalam beberapa menit

⚠️ Pemecahan Masalah Umum

  • Data tidak terbaca: Pastikan file tidak dilindungi password · Coba "Save As" dari Excel terlebih dahulu
  • Grafik tidak muncul: Refresh halaman (F5) lalu upload ulang
  • Kabupaten hilang: Periksa ejaan nama kabupaten di sheet kpop — harus sama persis dengan tahun lalu
  • Tahun tidak terdeteksi: Pastikan header kolom berisi angka tahun (2025, 2026, dst) bukan teks seperti "Tahun 2025"
  • Format berubah total: Upload file ke Claude dengan pesan "Format Excel Verval berubah, tolong sesuaikan parser"
